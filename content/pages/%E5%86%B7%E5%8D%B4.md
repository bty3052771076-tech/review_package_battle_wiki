**冷却** (CD) 是所有有规律地触发的[物品](pages/%E7%89%A9%E5%93%81.html)所拥有的[游戏机制](pages/%E6%B8%B8%E6%88%8F%E6%9C%BA%E5%88%B6.html)。

在战斗中，物品会以冷却速度逐渐填充高亮。一旦完全填满，物品就会激活，冷却时间重置。

只能激活一次的物品，例如背包中放置的大多数[宝石](pages/%E5%AE%9D%E7%9F%B3.html)、[卡牌](pages/%E5%8D%A1%E7%89%8C.html)，一旦激活就不会重置，但它们的冷却时间可以像其他物品一样受到影响。

物品具有1000%（十倍）冷却速度的上限，即500层以上[![Image 1: Heat](../assets/images/b7be4cfac26858bedd57434ecac91dca0930faa68bb21da355f21bbb2dcd4c98.png)](pages/%E7%8B%82%E7%83%AD.html "狂热") [狂热](pages/%E7%8B%82%E7%83%AD.html)无法使物品有更快的冷却。

影响冷却的因素
-------

计算公式
----

所有加减速的效果相加或相减，最终冷却由所有这种效果之和决定。

如果总和为加速，使用以下公式：

Cooldown=Base Cooldown 1 0 0%+Faster−Slower

当前冷却 = 基础冷却 / ( 100% + 加速效果 - 减速效果 ）

如果总和为减速，使用以下公式：

Cooldown=Base Cooldown*(1 0 0%+Slower−Faster)

当前冷却 = 基础冷却 x（ 100% + 减速效果 - 加速效果 ）

简单来说，正的加速线性提升冷却速度，而负的减速则是线性提升冷却时间。

### 示例

4 s 1 0 0%+(4 0%+3 0%+2 0%+1 0%)−(2 0%)=4 s 1 8 0%=4 s 1.8=2.2 2 s

4s / 100% + (40% + 30% + 20% + 10%) - (20%) = 4s / 180% = 4s / 1.8 = 2.22s

另一个例子：

*   [神圣护甲](pages/%E7%A5%9E%E5%9C%A3%E6%8A%A4%E7%94%B2.html)：基础冷却3s
*   2 x [迅捷手套](pages/%E8%BF%85%E6%8D%B7%E6%89%8B%E5%A5%97.html)： 2 x +20% = +40%加速
*   50[![Image 2: 冰冷](../assets/images/cdb29718e7c8b69559d41cde9ef76a105e34d16fba210b61d4c7bded2c706037.png)](pages/%E5%86%B0%E5%86%B7.html "冰冷") [冰冷](pages/%E5%86%B0%E5%86%B7.html)： 50 x -2% = -100%减速
*   加速总计+40%，减速总计-100%，使用第二个公式，最终冷却时间为：

3 s*(1 0 0%+1 0 0%−4 0%)=3 s*1 6 0%=4.8 s

3s * (100% + 100% - 40%) = 3s * 160% = 4.8s
