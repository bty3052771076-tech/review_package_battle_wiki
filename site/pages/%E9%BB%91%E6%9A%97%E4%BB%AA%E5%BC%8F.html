黑暗仪式

听我解释，我们没有在举行什么邪恶的仪式。

数据

| 稀有度 | 特别 | | --- | | 类型 | 技能 | | 价格 | 5 | | 职业 | 收割者 | | 效果 | * 17s后：向对手施加20层减益效果，获得10层。 * 每有1个的物品，黑暗仪式的触发快20%。 | | 是否在商店 | 将在第10回合出现 |

格子

黑暗仪式是收割者的一个技能，会在第十回合出现。

购买后下一个腐败水晶出现概率x1.5权重。

使用技巧

  • 在有至少5个以上物品吃到延迟会在可接受范围。
  • 大量的减益效果可以配合诅咒匕首或黑暗剑。
  • 也能配合霜冻之剑。
