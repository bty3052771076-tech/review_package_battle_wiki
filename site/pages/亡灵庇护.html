亡灵庇护

就算你不庆祝亡灵节，它们也会庇护你的。

数据

| 稀有度 | 特别 | | --- | | 类型 | 技能 | | 价格 | 5 | | 职业 | 通用 | | 效果 | * 物品的最大生命值提升效果+20%。 * 过量治疗时：过量治疗的生命值增加为最大生命值。 * 每7s：恢复10点生命值，且每有一个的物品，生命值恢复+4。 | | 是否在商店 | 将在第3和第10回合出现 |

格子

亡灵庇护是一个技能。可以将过量治疗转化为最大生命值。

使用技巧

  • 对于大量回血的流派是很大的提升。比如具有大量/的，及鹿木守护等。
  • 使用红宝石进一步提升回血量。
