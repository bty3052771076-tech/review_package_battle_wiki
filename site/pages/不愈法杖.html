不愈法杖

想象一下被治愈的感觉，但是相反，并且糟糕很多倍。

数据

| 耐力消耗 | 1.3 | | --- | | 稀有度 | 神级 | | 类型 | 武器 | | 价格 | 17 | | 职业 | 收割者 | | 效果 | 每2s：恢复25生命值。消耗4点：2s内，向对手造成你治疗量100%的伤害。 | | 是否在商店 | 获得前提彩虹魔药 |

格子

合成配方

  • 法杖 + 恶魔烧瓶
  • 价格 17, (10+7)

不愈法杖是一个特殊的武器，不会进行攻击而是获得不治效果。

不愈法杖会在激活后判断是否达到4点，如果达到4点会获得持续2秒100%的不治效果。对对手造成你所有治疗量（吸血，恢复，香蕉等提供的）的伤害，如果没有4点只会治疗你。由于不愈法杖没有实际上进行攻击，它不能被像磨刀石这样的物品加伤，也不会被吸血鬼梳子赋予生命偷取效果。

使用技巧

  • 使用红宝石等物品最大化恢复，也能抵消对手的恢复减少效果。
  • 物品如吸血鬼梳子/强力吸血魔药也可以获得大量的回复。
