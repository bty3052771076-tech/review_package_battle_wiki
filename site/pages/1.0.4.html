大家好，

我们这次的目的是让不同的子职业更加平衡。

对于魔法师，"打造卷轴"和"国际象棋"系列物品已经有了一些起色，但这还不够，它们这次获得更多的加强。


我们会持续进行每周的平衡更新，直到所有1.0的新内容都能顺利融入为止，之后会逐渐放缓更新节奏。


祝大家玩得开心！：)

1.0.4 版本更新日志

中立

  • 魔法球： 增益 22 -> 20
  • 极乐小鸟： 添加"自然"词条
  • 血之黏黏： 初始吸血 2 -> 3

游侠

  • 雪人大师： 冷却 3.3s -> 3.1s
  • 穿甲箭： 幸运获取几率 65% -> 60%
  • 王牌橡子： 耐力降低 12% -> 10%
  • 女射手： 攻击速度 30% -> 35%

收割者

  • 剧毒蛙： 冷却 3.8s -> 3.4s
  • 抱抱先生： 速度加成 110% -> 100%
  • 吸血鬼梳子： 恢复效果 25% -> 30%

火焰魔导士

  • 黯淡提灯： 无敌时间 1.5s -> 1.3s
  • 燃烧旗帜： 致盲几率 50% -> 65%

狂战士

  • 尖刺墙： 伤害格挡 12 -> 15
  • 旅行包： 减伤 22% -> 25%
  • 鹿木守护： 减伤 10% -> 12%
  • 萨满面具： 冷却 3.4s -> 3.2s
  • 龙骑士： 冷却 1.1s -> 1.0s

魔法师

  • 吞噬圆球： 冷却 3.7s -> 3.2s
  • 死亡莲花： 增益移除 2 -> 3
  • 浆果提灯： 冷却 4.2s -> 3.4s
  • 霜冻花： 寒冷几率 25% -> 30%, 护盾 13 -> 17
  • 彩虹魔棒： 冷却 6s -> 5.5s
  • 魔法入门指南： 最大生命加成 7 -> 8
  • 魔法杯糕黏黏： 治疗 12 -> 18
  • 棋盘： 冷却 0.8s -> 0.7s
  • 白色骑士： 耐力获取 1 -> 1.5, 速度加成 35% -> 50%
  • 黑色骑士： 耐力移除 1 -> 1.5
  • 白色主教： 净化 6 -> 8, 恢复效果提升 10% -> 15%
  • 黑色主教： 减益施加 4 -> 5, 恢复效果/最大生命获取削弱 10% -> 15%
  • 白色城堡： 护盾 30 -> 35
  • 黑色国王： 价格 6g -> 5g
  • 白色国王： 价格 6g -> 5g
  • 水元素： 冷却 3.2s -> 2.8s
  • 终极升腾： 增加联动星位. 冷却 12s -> 10s, 速度加成 15% -> 25%

冒险家

  • 英雄盾牌： 伤害加成 13% -> 15%
  • 麻绳：持续时间 5s -> 6s
  • 水银元素： 护盾转化 25% -> 30%
