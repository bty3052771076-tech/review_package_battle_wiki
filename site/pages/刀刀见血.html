刀刀见血

七步之内，匕首又快又准。

数据

| 稀有度 | 特别 | | --- | | 类型 | 技能 | | 价格 | 5 | | 职业 | 通用 | | 效果 | * 匕首武器攻击速度快40%。 * 12s后：你的武器造成的伤害+50%。 * 每有一个的匕首，刀刀见血的效果触发快40%。 | | 是否在商店 | 将在第3回合出现 |

格子

刀刀见血是一个技能，可以大幅增幅武器的伤害。

匕首包括： 匕首， 剧毒匕首， 匕首回旋镖， 吸血匕首， 幽灵匕首， 炽热匕首， 诅咒匕首。

购买后下三把匕首出现概率x1.7权重。

使用技巧

  • 持有匕首可以使生效延迟在可用的范围内。对匕首的攻速加强也能提升进一步匕首的输出。
