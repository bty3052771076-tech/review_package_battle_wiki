0.9.4
日期 2024-4-5
← 0.9.3 0.9.5 →

大家好！

我们在4周前发布了《背包乱斗》，可以说这是一个巨大的成功！这个游戏对我们意义重大，所以我们会继续把它打造成最优秀的游戏 🤍

这次我们有更多的实验性调整！这一次更新添加了徽章，它们属于独特物品，可以在游戏过程中的任何时候出现，但它们会以较高的概率出现在第一回合（高达25%！）。它们的设计目的在于增加多样性，并为你提供一些在其他情况下无法实现的游戏体验。

我们添加了6个徽章，这些徽章都会影响你能够在商店购买的物品。让我们一起测试一下，我们会根据大家的喜爱程度来调整出现频率，如果我们都确认大家喜欢这个新功能，我们可能会添加更多的徽章。


此外，我们正在对Discord 和中文社区进行问卷调查。到目前为止，我们有三个主要收获： 大家想要一个高方差的职业——冒险者（路线图的新职业）将基于此进一步设计！ 大家想要更多的独特物品——我们将继续添加这些物品！ 大家想要在共通池中有更多的道具——我们已经开始设计了！


在平衡方面，游侠，收割者和狂战士似乎势均力敌，火焰魔导士的表现仍然很优秀，但差距不是很大。因此在这方面我们会更谨慎的调整。 这一次更新的主要目的是增强一下因为强度问题导致很冷门的转职和道具。 我们将继续每周更新，直到我们认为目前所有的转职都能在一些构筑里有可行性！

- Dorothea and Mario

新内容

新增6个独特品质的徽章：徽章会在所有回合作为独特物品出现，且它们会在第一回合有更高概率物品出现！（25%出现率）。同时游玩时，商店中不会出现自身职业的徽章。（例：狂战士不会找到狼徽章）

  • 绿叶徽章(游侠专属物品)
  • 骷髅徽章(收割者专属物品)
  • 火焰徽章(火焰魔导士专属物品)
  • 狼徽章(狂战士专属物品)
  • 彩虹徽章(允许购买所有职业的专属物品)
  • 石头徽章(不再刷出自己职业的专属物品)


平衡

  • 基础“独特物品出现率”提升至3%（其它独特物品仍然在回合4之后出现）
  • 影响平衡性的bug修复：修复了在耐力不足时被盾牌格挡的情况下，盾牌的耐力扣除效果失效的问题。基于该bug修复的平衡性补偿：所有盾牌的格挡率：35%->30%。（注：这个调整很难预测，后续会重点观察并进行相关调整。）

共通物品

  • 燃烧火把: 伤害 2-4 -> 2-3, 被动触发条件 8 -> 12
  • 荷包蛋圣锅: 冷却 1.5s -> 1.6s
  • 钢之黏黏: 16 -> 14
  • 笨重巨剑（被动效果调整）：耐力消耗 3 -> 2.5, 冷却 3s -> 2.5s
  • 酷炫西洋剑: 冷却 1.5s -> 1.4s, 耐力消耗 1.0 -> 0.8
  • 号角: 冷却 4.5s -> 4.0s
  • 生命魔药: 回复 11 -> 12
  • 强力生命魔药: 回复 18 -> 20
  • 吸血魔药: 吸取生命 10 -> 12
  • 白金会员卡: 出现率加成10% -> 15%
  • 心之容器/黑暗心之容器: 最大血量提升 80 -> 90
  • 力量手套: 4 -> 5
  • 石头: 冷却 3.5s -> 3.2s, 护甲移除 3 -> 4
  • 守护钱包: 7 -> 8
  • 菠萝: 回复 3 -> 4
  • 荆棘之鞭: 冷却 2.4s -> 2.2s, 耐力消耗 2.4 -> 2.2
  • 血棘: 冷却 1.7s -> 1.6s, 耐力消耗 1.8 -> 1.6
  • 饥饿之刃: 冷却 1.7 -> 1.6, 伤害 +1
  • 腐败熊猫之锅: 伤害 +2
  • 英雄长剑/十字剑/光剑/法杖: 伤害 +1


火焰魔导士

  • 火焰剑: 伤害 -1
  • 龙铭珠: 冷却 3.0s -> 3.5s
  • 日曜护甲: 回复 17 -> 15
  • 黑晶龙: 被动触发条件 22 -> 26
  • 黯淡提灯: 联动奖励 6 -> 7
  • 炽热匕首: 伤害 +2
  • 炽热长矛: 移除 4 -> 5, 额外伤害 3 -> 5
  • 烈火之鞭: 狂热 3 -> 4, 冷却 1.7s -> 2.0s
  • 烈火之杖: 伤害 +2


狂战士

  • 铜指虎: 冷却 2.0s -> 2.1s
  • 锁链鞭: 冷却 2.3s -> 2.1s, 耐力消耗 2.3 -> 2.1
  • 鹿木守护: 时间延长加成 0.5s -> 0.6s
  • 狼印记: 减益抵抗需求 5 -> 3
  • 所有符文（萨满面具转职）: 价格 5g -> 4g
  • 龙鳞靴: 8 -> 12,抵抗 25% - 20%
  • 锻造巨剑: 奖励 +4 -> +5
  • 锻造锤: 价格 6g -> 3g.（售价3g->2g）
  • 锻造锤: 允许从商店内购买（应用于狼徽章，彩虹徽章和手贱出售后进行购回）


游侠

  • 一碗零食: 双重触发几率 25% -> 20%, 食物加速 5% -> 10%
  • 世界树之叶: 回复 8 -> 10
  • 穿甲箭: 移除 5 -> 8
  • 天命短弓: 获得几率 60% -> 70%, 基础命中 85% -> 100%
  • 颠茄剧毒短弓: 施加率 70% -> 60%, 施加层数 1 -> 2
  • 颠茄剧毒弓: 施加增长 9 -> 8
  • 尖牙弓: 起始 2 -> 3
  • 萝卜头黏黏: 持续时间 4s -> 5s

收割者

  • 蛇蛇: 冷却 2.2s -> 2.6s
  • 命运小姐: 冷却 2.0s -> 1.8s
  • 大煮锅（触发效果变更）: 回复12生命，并获得4或3
  • 大煮锅：新增“”标签
  • 愚者: 价格 5g -> 4g
  • 小丑牌: 价格 6g -> 5g
  • 不愈法杖: 回复 +1
  • 红宝石巨龙: 伤害 +2
  • 吸血鬼梳子（重做）: 武器获得生命偷取，额外的物品提供偷取比例提升
  • 贴贴女士（新增被动）: 每个物品提供10%速度, 冷却: 3s -> 3.5s
  • 强力恶魔烧瓶（新增触发条件）: 生命值低于 25% 时也会触发

其他变更

  • 物品的触发优先级变更：不再基于购买顺序触发，而是基于新增的内置优先级触发,如提灯复活会优先于其他复活触发。（如果优先级相同，则触发先后顺序随机。）
  • 游戏已支持“steam增强版丰富状态”：你能够看到你的好友在游戏里做什么了！

感谢社区玩家旋律星萤协助翻译此公告！
