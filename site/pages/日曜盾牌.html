日耀盾牌

盾牌中间部分用来煎鸡蛋非常合适。

数据

| 稀有度 | 神级 | | --- | | 类型 | 盾牌 | | 价格 | 14 | | 职业 | 火焰魔导士 | | 效果 | * 的物品每获得12点：造成4点伤害。 * 被攻击时（/）：有30%的几率抵挡17点伤害并移除对手0.7耐力。 | | 镶嵌位 | 2 | | 是否在商店 | 否 |

格子

合成配方

  • 勇气盾牌 + 火焰 + 火焰
  • 价格 14(12+1+1)

日耀盾牌是一个盾牌。

相比勇气盾牌还能抵挡攻击并高3点格挡量（但是造成伤害效果替代了增幅）。
