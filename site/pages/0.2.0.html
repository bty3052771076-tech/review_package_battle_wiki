补丁说明

  • 新增吸血护甲（可制作）
  • 新增勇气盾牌
  • 添加了每场战斗后显示背包的按钮

修复：

  • 修复了极少数情况下匹配系统在早期回合匹配到后期对手的问题
  • 修复了光剑消耗护盾而非致盲效果的问题
  • 致盲之光现在使用正确的持续时间（3秒而非4秒）
  • 修复了剧毒匕首被眩晕触发时不施加中毒层数的问题
  • 石头和红宝石幼龙现在被标记为武器
  • 修复了带有"致盲"的提示框同时显示"护盾"上下文提示的问题
  • 修复了笛子/号角视觉效果不激活的问题
  • 修复了在排名加载完成前打开商店时发生的显示错误
  • 改进了文件处理以更好地应对损坏的存档
  • 修复了尖刺盾牌触发过于频繁的问题
  • 修复了物品提示框仍显示旧背包效果的问题
  • 修复了物品出售后瞬间仍可悬停的问题
  • 将有效鼠标位置限制在游戏窗口内以防止异常情况
  • 修复了离开商店后仍可购买物品导致的崩溃和存档损坏问题
  • 修复了上一回合过快点击关闭时庆典号角闪光粒子显示异常的问题
  • 面向下方或左侧的物品进度着色器现在显示正确
  • 修复了继续游戏时红宝石蛋过早孵化的问题
  • 修复了战斗结束后仍可调整时间缩放滑块导致游戏速度异常的问题
  • 修复了荆棘鞭不应用伤害加成的问题

平衡性调整:

  • 光剑:降低致盲之光层数10→7
  • 饥饿之刃:吸血转化从2降至1，现在开局获得1层吸血，伤害-1
  • 血棘:吸血转化从3降至2，伤害降至3-6
  • 恶魔烧瓶:每层减益伤害1.0→0.7
  • 木剑:最大伤害+1
  • 勇者之剑:最大伤害+1
  • 长矛:最大伤害+1，现在移除4点护盾
  • 平底锅:最大伤害+1
  • 匕首/剧毒匕首:最大伤害+1
  • 储物棺材:触发几率35%→25%
  • 尖刺盾牌:现在每获得15点护盾生成1根尖刺（原为10点）
  • 号角:冷却时间5秒→3秒
  • 笛子:现在随机提供10点护盾、2点耐力或2点幸运值
  • 弓箭:冷却时间4秒→3.5秒
  • 锯刃:耐力消耗2→1.5
  • 荆棘之鞭:现在暴击时获得+1根尖刺
  • 大蒜:护盾值5→4
  • 神圣护甲:中毒清除冷却时间3秒→2秒
  • 致幻蘑菇:售价2金→3金

其他改动：

  • 多项提示框优化！文本更统一，冗余信息更少，高亮和布局更合理，现在会高亮触发条件...
  • "受影响"星标现在会显示在背包区域外的槽位上
  • 物品现在会显示它们在背包区域外所占用的槽位
  • 商店中悬停物品时会显示其尺寸和星标（影响的槽位）
  • 排名加载速度大幅提升！
  • 商店中物品重叠顺序更合理（下方的物品在前端显示）
  • 为背包能量添加悬停提示框
  • 商店音乐现在能正确循环播放
