献祭之碗

旅行时也能随机进行献祭！

数据

| 稀有度 | 特别 | | --- | | 类型 | 背包 | | 价格 | 8 | | 职业 | 火焰魔导士 | | 效果 | * 增加4格背包槽位。 * 战斗开始时：获得1层。 进入商店时：献祭碗内所有物品。生成一团火焰，还会根据献祭物品的总价值生成其他物品。 | | 是否在商店 | 否 |

格子

献祭之碗是火焰魔导士的一个初始背包。

说明

献祭之碗上会显示碗内物品的总价值，如果一共价值X，那么下回合进入商店时，会生成1个火焰，和其他总价格为X-1的物品。

献祭之碗只能生成可以在商店中购买到的物品。

使用技巧

  • 可以将不需要的物品转化掉
  • 将打折的物品转化掉可能可以换为有用的物品。
  • 献祭7的物品来获得6的物品，其中大部分为各种护符，在火焰魔导士的各种流派中有重要作用。
