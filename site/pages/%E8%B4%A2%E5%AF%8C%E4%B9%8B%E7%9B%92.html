财富之盒

有时候，盒子本身比装的内容更珍贵。

数据

| 稀有度 | 史诗 | | --- | | 类型 | 背包 | | 价格 | 5 | | 职业 | 通用 | | 效果 | * 增加4背包槽位。 * 进入商店时：如果有2个神级/特别物品在盒子里产生一个宝石碎片。 | | 是否在商店 | 否 |

格子

合成配方

  • 宝石盒 + 幸运招财猫
  • 价格 5

财富之盒是一个背包，可以生成宝石碎片。

合成之后商店中无法再刷新出宝石，想要继续购买宝石的话需要确保还有额外的宝石盒。
