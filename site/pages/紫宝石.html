紫水晶

传说紫水晶能让人保持清醒，其实是有道理的，买了紫水晶就没钱买酒了。

数据

| 稀有度 | 多种 | | --- | | 类型 | 宝石 | | 价格 | 1/2/4/8/16 | | 职业 | 通用 | | 效果 | 放入武器槽： 击中对手： 将有25/35/50/75/100% 的几率移除对手2个随机增益效果。 放入护甲或其他槽： 对手的生命值恢复效果降低12/18/25/30/45%。 放入背包： 每3.2/2.5/2/1.6/1s：净化自身1个随机减益效果。 | | 是否在商店 | 获得前提宝石盒 |

格子

拥有 宝石盒/宝石存钱罐/财富之盒可以获得的五种宝石之一。

紫水晶有五种品质，分别是：碎片/劣质/普通/优质/完美，对应游戏中的物品五种稀有度。每种稀有度的宝石都比前一种稀有度具有更好的属性。如果你有宝石盒，可以在商店找到更高稀有度的宝石，或者通过在背包中合成相同的宝石来获得。

宝石的效果因放置位置的不同而有所差异。
