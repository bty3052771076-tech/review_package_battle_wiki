火焰徽章

我买的时候以为是打火机。

数据

| 稀有度 | 特别 | | --- | | 类型 | 配饰 | | 价格 | 5 | | 职业 | 通用 | | 效果 | * 可获得火焰魔导士物品。 * 进入商店时：有65%的几率花费1枚来生成一团火焰。 * 开始战斗时：获得6层。 | | 是否在商店 | 是 |

格子

火焰徽章是一个宝藏物品。将会在第1到第18回合出现。

使用技巧

  • 开局6层全局12%加速十分有性价比。
  • 很适合“炒火”（快速累积大量）的阵容。
