蓝宝石

传说蓝宝石会让土星给你带来好运，其实是扯淡。

数据

| 稀有度 | 多种 | | --- | | 类型 | 宝石 | | 价格 | 1/2/4/8/16 | | 职业 | 通用 | | 效果 | 放入武器槽： 击中对手：将有15/25/40/60/100%的几率无视直接攻击，并获得1点还会向对手施加1层。 放入护甲或其他槽： 每获得5点：将获得2/3/4/5/8点。 放入背包： 4.5/4.5/4/3.5/3s后：向对手施加2/3/4/5/8层。 | | 是否在商店 | 获得前提宝石盒 |

格子

蓝宝石是拥有 宝石盒/宝石存钱罐/财富之盒可以获得的五种宝石之一。

蓝宝石有五种品质，分别是：碎片/劣质/普通/优质/完美，对应游戏中的物品五种稀有度。每种稀有度的宝石都比前一种稀有度具有更好的属性。如果你有宝石盒，可以在商店找到更高稀有度的宝石，或者通过在背包中合成相同的宝石来获得。

宝石的效果因放置位置的不同而有所差异。
