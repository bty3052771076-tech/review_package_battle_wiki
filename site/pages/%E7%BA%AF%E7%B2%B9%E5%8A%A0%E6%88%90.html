纯粹加成

一点加成不算什么，但是积少成多。

数据

| 稀有度 | 特别 | | --- | | 类型 | 技能 | | 价格 | 5 | | 职业 | 通用 | | 效果 | 战斗开始时：获得10%最大生命值，10%基础耐力恢复。 | | 是否在商店 | 将在第3回合出现 |

格子

纯粹加成是一个保底技能，必定会在第三回合出现。

10%的耐力恢复仅相当于黄宝石碎片，10%的额外生命值前期不到10点。这使得一般情况下不如未挖掘潜力省一点赌一下。
