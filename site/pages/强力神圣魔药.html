强力神圣魔药

小抿一口就微醺。

数据

| 稀有度 | 神级 | | --- | | 类型 | 魔药 | | 价格 | 15 | | 职业 | 收割者 | | 效果 | 如果你的减益效果达到10层：消耗此魔药并净化10层减益效果，还会获得8层随机增益效果。 | | 是否在商店 | 否 |

格子

合成配方

  • 神圣魔药 + 大煮锅
  • 价格 17 (7+10)

强力神圣魔药是一个魔药。

使用技巧

  • 可以获得8层随机增益效果，通过其他魔药顶总计可以获得24层效果。
    • 使用容易开局触发的魔药顶如硬化魔药/瘟疫烧瓶可以更快获取增益。
